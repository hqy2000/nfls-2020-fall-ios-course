## Assignment 2 `VStack` 及 `HStack` 排版复习

1. 去百度搜索上下载学校校徽，并使用 `Image` 添加至屏幕左上角，图片大小要需要适中。
2. 在校徽右侧，使用 `Text` 书写“南京外国语学校”及英文“Nanjing Foreign Language School“。英文需要在中文的下方，且字体要小一些。这两步需要嵌套 `HStack` 与 `VSatck`。
